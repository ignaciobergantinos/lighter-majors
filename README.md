# lighter-majors

A fast, keyboard-driven trading client for the [Lighter Exchange](https://lighter.xyz) — built for executing perpetual futures trades on majors (BTC, ETH, SOL) with minimal friction.

Place market orders, manage open positions, and auto-size trades as a percentage of available balance. Comes with per-coin preferences, Discord fill notifications, sound cues, and global keyboard shortcuts in the desktop build.

Available as both a Next.js web app and an Electron desktop app.
